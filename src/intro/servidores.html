CodeSaur - Servidores

  Websites/Webpages são basicamente arquivos armazenados em computadores que executam um pedaço de software conhecido como web server (Apache, Nginx, etc)

  Empresas de hospedagem são responsáveis por providenciar espaços nos servidores para o seu site

  Domínios podem ser adquiridos através de registradores e vinculados ao seu hospedeiro.

  DNS é utilizado para mapear endereços de IP.
